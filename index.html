    E-Commerce Cost Engine & Financial Reporting Pipeline

    Executive Summary

    Developed an end-to-end data automation ecosystem for a multi-channel e-commerce firm managing 8,000+ SKUs and 20+ manufacturers. Transformed a manual accounting process into a scalable $60/month cloud-native solution.

    The Challenge

    Variable pricing across 20 manufacturers, multi-channel marketplace commission complexity, logistical fragmentation, and a resource drain of 80 man-hours per cycle across 3 senior staff over 5 days.

    Technical Solution

    Azure ETL/ELT pipeline with custom API retrieval platform, multi-format Data Lake handling JSON, CSV, and Parquet, star schema SQL Data Warehouse, Power App for logistics rate updates, Google Sheets-to-Azure pipeline for business users, and automated Paginated Reports generating client-ready invoices.

    Business Impact

    93.7% labor reduction. Turnaround time cut from 5 days to 2 days. Operational cost from 3 senior FTEs to $60/month. 75 hours of high-level employee time reclaimed per cycle.

    Technology Stack

    Azure Data Factory, ADLS Gen2, MSSQL, Power BI, Power Apps, Amazon SP-API, Shopify API.

    Industrial Scale-Up & Operational Intelligence — Mining Operations

    Executive Summary

    Integrated data ecosystem for a rapidly expanding mining and raw materials producer following a series of acquisitions. Automated document processing, unified task management with financial flows, and provided C-level executives with real-time decision-making snapshots.

    The Challenge

    Non-standardized processes across mining sites, data trapped in local computers and manual invoices, and massive human effort required to map cash flows to mining tasks.

    Technical Solution

    Azure Data Factory and Databricks for ETL/ELT. Azure Machine Learning OCR to digitize manual invoices and map them to project tasks. ClickUp API integration linking every task to its financial cash flow. Power Apps for structured data entry. SharePoint as a governed document library. Row-Level Security and Active Directory integration.

    Business Impact

    700% increase in monthly task capacity. 314% increase in raw material processing. 40% reduction in labor inefficiency. Data accessibility from under 10% to 80% organization-wide. Total cost of ownership €15,000 over two years.

    Technology Stack

    Azure Databricks, Azure Data Factory, Azure Machine Learning, Power BI, Power Apps, SharePoint, ClickUp API.

    Strategic Architectural Overhaul & Data Integration — E-Commerce Platform Recovery

    Executive Summary

    Forensic audit and complete architectural overhaul of a failing e-commerce PHP monolith operating across Amazon and Shopify. Replaced with a modern decoupled microservices cloud architecture.

    The Challenge

    Single massive codebase caused regressions across modules. Only the original author could navigate the system. Reports hard-coded in Python. Single-threaded ETL lagged as sales volume grew.

    Technical Solution

    Azure Data Factory for orchestration and ETL. Azure Databricks for Spark compute. Azure SQL as Single Source of Truth. Power BI for self-service BI. Azure DevOps CI/CD separating Dev, Staging, and Production. ADLS Gen2 Bronze layer for raw JSON. Terraform and Bicep Infrastructure as Code. Azure AD RBAC for department-level access control.

    Technology Stack

    Azure Data Factory, Azure Databricks, Azure SQL, ADLS Gen2, Power BI, Azure DevOps, Terraform, Bicep, Azure Active Directory, ClickUp, Amazon SP-API, Shopify API.

    Precision Viticulture: End-to-End IoT & Irrigation Ecosystem

    Executive Summary

    Complete precision agriculture system for a 12-hectare vineyard combining IoT sensor infrastructure, edge computing, cloud data pipelines, and predictive automation for fully autonomous weather-aware irrigation.

    Technical Solution

    IoT devices monitoring soil NPK, moisture, and temperature in real-time. Low-power always-on on-premise server as data retrieval center. Azure and Databricks pipelines syncing sensor data with Weather Forecast APIs. Machine learning model forecasting water needs based on evapotranspiration versus predicted rainfall. Automated pump control maintaining optimal moisture while minimizing energy.

    Technology Stack

    IoT soil sensors, on-premise edge computing server, Azure cloud pipelines, Azure Databricks, Weather Forecast API, automated pump control systems.

    Predictive Infrastructure & Operational Optimization for Urban Water Utility

    Executive Summary

    Turned a reactive century-old water utility serving 3M+ residents into a predictive data-driven operation. Daily incident throughput doubled, average fix time dropped 33%, high-severity pipe bursts fell 12% year-over-year.

    Technical Solution

    PostgreSQL data warehouse consolidating SAP incident records, IoT telemetry from pressure gauges and reservoir levels, asset inventory, and billing consumption map. Machine learning forecasting achieving 80% daily, 90% weekly, and 93% monthly confidence. KPI and alert management layer connected back into SAP via custom REST API generating work orders. Power BI dashboards for incident volume, fix-time distribution, team utilization, and warehouse readiness.

    Technology Stack

    SAP ERP, PostgreSQL, Python ML, Power BI, IoT telemetry, custom REST APIs, Azure infrastructure.
